# Crime and Weather Data Analysis (R)

## Overview
This project analyses the relationship between crime patterns and weather variables using R.
It was completed as coursework and demonstrates exploratory data analysis,
visualisation, and statistical interpretation using R Markdown.

## Tools & Technologies
- R
- R Markdown
- ggplot2
- dplyr

## How to Run
Open the `.Rmd` file in RStudio and click **Knit** to generate the report.

## Key Insights
- Identified seasonal and weather-related trends in reported crime incidents.
- Observed correlations between specific weather variables and crime frequency.
- Demonstrated the use of time-series visualisation for pattern interpretation.
